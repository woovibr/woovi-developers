## Instale o Plugin woovi na sua instância WooCommerce

Este documento detalha passos necessários para conectar a sua plataforma de e-Commerce, baseada no WooCommerce, na woovi.

### 1. Instale a woovi na sua instância WooCommerce

[woovi For WooCommerce](https://wordpress.org/plugins/woovi-for-woocommerce/)

![Banner](/img/ecommerce/woocommerce-banner.png)
![Install](/img/ecommerce/woocommerce-plugin-install.png)

### 2. Configurando o Plugin WooCommerce

Ao configurar uma nova integração, será possível criar novos pedidos com woovi Crediário.

É necessário ter um [AppID](../../plugin/app-id.md) do tipo _Plugin_ para que seja possível a integração.

Para tanto, entre na plataforma e crie uma nova aplicação na página de [_Nova API/Plugin_](https://app.woovi.com/home/applications/add) em _API/Plugins > Nova API/Plugin_:

![Criando uma nova aplicação na plataforma](./__assets__/woocommerce-pix-crediary-platform-new-application.png)

Ao ser redirecionado para a página de visualizar uma aplicação, copie o _App ID_:

![Copiando o identificador da aplicação](./__assets__/woocommerce-pix-crediary-platform-application-id.png)

Vá até a página de configurações do seu plugin, clicando em _Settings Pix Crediary_, para configurar o plugin:

![Configurações Settings Crediário](./__assets__/woocommerce-plugin-settings-v2100.png)

Cole o _App ID_ copiado no campo _AppID woovi_:

![Colando o identificador da aplicação](./__assets__/woocommerce-pix-crediary-plugin-settings-app-id.png)

### 3. Ativando Plugin Brazilian Market

Para utilizar o woovi Crediário e salvar o customer da order na sua cobrança woovi é necessário que seja ativado um plugin que possibilite que o cliente informe os campos abaixo no momento da compra.

- Nome
- Cpf/Cnpj
- Email
- Telefone
- Endereço

Indicamos o uso do plugin [woocommerce-extra-checkout-fields-for-brazil](https://wordpress.org/plugins/woocommerce-extra-checkout-fields-for-brazil/) para este caso. Uma vez ativado o plugin passará a salvar os clientes que informarem o CPF/CNPJ no momento da compra.

### 4. Configurações do Plugin no WooCommerce

Ao acessar as configurações do plugin após os passos anteriores note que o AppID woovi ja estará preenchido e você poderá editar o título, descrição e label do botão de compra do plugin.

Para o tutorial iremos seguir com o titulo **Pagar com woovi Crediário** como no print abaixo:

![Configurações do Pix Crediário](./__assets__/woocommerce-pix-crediary-plugin-settings.png)

## 5. Realizar Pedido com Pix Crediário no WooCommerce

Escolha a opção de pagar o pedido usando woovi Crediário ou o nome que você escolheu no momento da configuração do plugin.

![Woocommerce Checkout Crediário](./__assets__/woocommerce-pix-crediary-checkout.png)

Selecione a opção de pagamento:

![Selecionando a opção de pagamento](./__assets__/woocommerce-pix-crediary-selecting-payment-option.png)

Pague a entrada via Pix usando o app do seu banco:

![Pague o QR Code](./__assets__/woocommerce-pix-crediary-payment.png)

## 4. Como ver meus logs woovi em minha loja WooCommerce/Wordpress

Você pode visualizar os logs woovi em sua loja através dos seguintes passos:

- 1. Tenha acesso aos arquivos da loja
- 2. Acesse o seguinte path: `/wp-content/uploads/wc-logs`
- 3. Dentro de `wc-logs` você irá encontrar os arquivos de logs da woovi com o seguinte padrão de nome `woocommerce_woovi-2023-01-13-7d609d821235742dd8162bbb0ef84862`
