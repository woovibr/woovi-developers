## Buscar Disputa (GET)

O endpoint GET de `Disputa` retorna um objeto com os seguintes campos:

- `dispute`: Detalhes da disputa

### Campos do objeto `dispute`:

- `type`: Tipo da disputa (`MED` ou `CHARGEBACK`)
- `status`: Status da disputa
- `endToEndId`: Identificador end-to-end da transação
- `id`: ID único da disputa
- `name`: Nome do cliente que abriu a disputa (apenas para `type: CHARGEBACK`)
- `email`: Email do cliente (apenas para `type: CHARGEBACK`)
- `phoneNumber`: Telefone do cliente (apenas para `type: CHARGEBACK`)
- `value`: Valor da disputa em centavos (apenas para `type: CHARGEBACK`)
- `disputeReason`: Motivo da disputa (apenas para `type: CHARGEBACK`)

### Tipos de disputa:

- `MED`: Mecanismo Especial de Devolução - Retorna campos básicos da disputa
- `CHARGEBACK`: Contestação de pagamento na Woovi - Retorna campos detalhados incluindo dados do cliente

## Exemplos de Resposta

### Disputa do tipo MED

```json
{
  "dispute": {
    "type": "MED",
    "status": "REJECTED",
    "endToEndId": "E31680151202503252045C0MHIPSJI6S",
    "id": "67e3170b38e62d5935ed4faf"
  }
}
```

```json
{
  "dispute": {
    "status": "ACCEPTED",
    "name": "John Doe",
    "email": "john.doe@example.com",
    "phoneNumber": "+5511999999999",
    "value": 10000,
    "disputeReason": "Product not received",
    "endToEndId": "Ea9c291526ae54b4cb41d9909bdf6d792",
    "type": "CHARGEBACK"
  }
}
```
