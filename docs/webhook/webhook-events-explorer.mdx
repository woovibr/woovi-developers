# Explorador de eventos de webhook

Selecione um tipo de evento na barra lateral e copie o payload em **JSON**, **TypeScript types**, **JSON Schema**, **Yup** ou **Zod**.
Os tipos e schemas são inferidos automaticamente a partir do payload de exemplo.
