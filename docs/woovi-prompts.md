[Woovi Prompts](https://github.com/woovibr/woovi-prompts) é um repositório
público com prompts prontos para usar com LLMs e agentes (como Claude Code,
Cursor, Copilot e similares) ao integrar com a plataforma Woovi.

Use os prompts para acelerar:

- Geração de cobranças via API Woovi
- Integração com Pix, cobv, boleto e subscriptions
- Tratamento de webhooks
- Setup de plugins e SDKs
